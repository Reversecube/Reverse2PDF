== Frequently Asked Questions ==

= How do I create my first PDF template? =

1. Go to Reverse2PDF â†’ Add New
2. Use the visual builder to add elements
3. Configure your template settings
4. Save and test your template

= Can I integrate with my contact forms? =

Yes! Reverse2PDF supports 10+ major form plugins including Contact Form 7, Gravity Forms, WPForms, and more.

= How do I add dynamic content to PDFs? =

Use our shortcode system or field mapping. You can display form data, post content, user information, and more using simple placeholders like {field_name}.

= Does it work with WooCommerce? =

Absolutely! You can create order confirmations, invoices, shipping labels, and more using WooCommerce data.

= Can I customize the PDF appearance? =

Yes! You have full control over fonts, colors, layouts, headers, footers, and more through the visual builder.

= Is there a limit on PDF generation? =

No limits! Generate as many PDFs as needed. We recommend enabling caching for optimal performance.

= Can I send PDFs via email automatically? =

Yes! Configure email delivery in your form integrations to automatically send PDFs to users or admins.

= Does it support multiple languages? =

Yes, the plugin is translation-ready and supports RTL languages.
